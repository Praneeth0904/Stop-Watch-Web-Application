## 📌 Stop Watch Web Application

An interactive stopwatch that dynamically works based on users' inputs.

---

## 🛠️ Tech Stack

## ✨ Features

* Display time with interactive "Start", "Stop", and "Reset" buttons.
* Provides an easy and engaging user experience.
